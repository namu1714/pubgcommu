# 배틀그라운드 킬내기 커뮤니티

### 기능
+ 배틀그라운드 킬내기 실시간 중계 커뮤니티
+ 4vs4 팀 대결 가능(플레이어별로 킬, 데스, 치킨, 정지 집계)
+ 참여하는 BJ 정보 관리

### 사용 기술
> Spring Boot, JPA, React, WebSocket
